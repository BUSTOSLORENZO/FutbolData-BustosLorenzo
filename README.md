# FutbolData-BustosLorenzo

Proyecto web desarrollado con Django 

## Funcionalidades

La página permite gestionar información de:

- Jugadores
- Clubes
- Ligas

Se implementa:

- CRUD completo para cada modelo
- Buscador de elementos en la base de datos
- Sistema de login y logout
- Panel de administración de Django

## Orden para probar la aplicación

1. Ejecutar el servidor

python manage.py runserver

2. Abrir en el navegador

http://127.0.0.1:8000/

3. Navegar por las secciones:

- Inicio
- Jugadores
- Clubes
- Ligas
- Buscar

4. Panel de administración:

http://127.0.0.1:8000/admin

usuario: admin  
password: 123
